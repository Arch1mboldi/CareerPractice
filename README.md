# 华理C语言程序设计指导

针对华东理工大学计算机大类第一学期的课程《C语言程序设计》，本项目提供了非官方的额外指导。

### 预期内容

1. 对C语言初学者友好的、具体形象的图形概念解释
2. 该课程重点实验题目的补充测试样例与代码详解
3. 讨论板功能，主要用于代码交流
4. 对入门者实用工具的介绍
